#To-Do List App

Простий та зручний застосунок для керування твоїми задачами.
Додавай, видаляй і відмічай виконані справи — тримай свій день під контролем 🚀

## 🧠 Можливості

* ➕ Додавання нових задач
* ✅ Позначення задач як виконаних
* ❌ Видалення задач
* 🔄 Не відновлюеться після перезавантаження сторінки

## 🖥️ Як користуватись

1. Введи задачу у поле вводу
2. Натисни кнопку **"Додати"** або Enter
3. Клікни по задачі, щоб позначити її виконаною
4. Натисни ❌, щоб видалити задачу

## 🛠️ Технології

* **HTML** — структура
* **CSS** — стилі
* **JavaScript** — логіка та взаємодія

## 🚀 Запуск проєкту

###Онлайн

Перейди за посиланням у репозиторії та користуйся застосунком прямо в браузері
